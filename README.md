# Opis usługi medycznej - Teleporada Medyczna

Nasza usługa teleporady medycznej umożliwia pacjentom szybki dostęp do konsultacji lekarskich bez wychodzenia z domu.  
Każda teleporada realizowana jest przez wykwalifikowany personel medyczny przy użyciu bezpiecznej platformy online.  
Dzięki usłudze pacjenci mogą uzyskać specjalistyczną opinię, zalecenia oraz niezbędne e-recepty podczas jednego połączenia.

# Jak pobrać repozytorium
**pobieranie repo**
1. Przycisk "code", kopiowanie ścieżki (np SSH) ,
2. w terminalu - > git clone nazwa ścieżki

 **Wyświetlanie gałęzi**
 - git branch -a
 - git checkout nazwa -> tutaj zmieniamy gałąź

**Przywracanie plików**
- git log - > namierzamy commit
- git reset --hard HEAD~1- > cofamy się o commit
- dodajemy do staging area a potem commitujemy
  
# Wybrane przydatne polecenia przy zakładaniu repozytorium
1. git init - > stworzenie nowego repozytorium
2. git add . - > przeniesienie wszystkiego co edytowane/utworzone do staging aera
3. git log --oneline - > okrojone wyświetlenie historii commitów
4. git commit -m "tekst" - commitowanie
5. git merge lub rebase - w zależności czy przenosimy z commitem, czy robimy porządki 
6. git checkout ,branch, - > zmiana gałęzi (też commita)
